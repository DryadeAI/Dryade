# Discord

The Dryade Discord server is the best place for real-time help, discussion, and community interaction.

## Join

**[discord.gg/bvCPwqmu](https://discord.gg/bvCPwqmu)**

## Channels

| Channel | Purpose |
|---------|---------|
| **#general** | General discussion about Dryade |
| **#help** | Technical help and troubleshooting |
| **#plugins** | Plugin development discussion |
| **#announcements** | Release notes and project news (read-only) |
| **#showcase** | Share what you've built with Dryade |

## Guidelines

- **Search before asking** -- Your question may already be answered in the channel history or [GitHub Discussions](https://github.com/DryadeAI/Dryade/discussions)
- **Share context** -- Include error logs, your OS, Docker version, and LLM provider when asking for help
- **Be specific** -- "X doesn't work" is hard to debug. "When I run `docker compose up`, I get error Y" is much better
- **Be patient** -- Community members are volunteers across different time zones

## Bug Reports

For reproducible bugs, please open a [GitHub Issue](https://github.com/DryadeAI/Dryade/issues/new) instead of posting on Discord. Issues are tracked, searchable, and easier to follow up on.

## Security

If you discover a security vulnerability, **do not post it on Discord**. Please report it privately via [security@dryade.ai](mailto:security@dryade.ai). See our [Security Policy](https://github.com/DryadeAI/Dryade/blob/main/SECURITY.md).
