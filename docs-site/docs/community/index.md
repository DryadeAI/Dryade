# Community

Welcome to the Dryade community. Whether you're building your first agent or deploying Dryade across your organization, you'll find help, inspiration, and collaborators here.

## Discord

Our primary community hub for real-time discussion and support.

- Get help with setup and configuration
- Share what you're building
- Discuss features and roadmap
- Connect with other Dryade users and contributors

**[Join Discord](https://discord.gg/bvCPwqmu)**

## GitHub Discussions

For longer-form conversations, questions, and ideas.

- **Q&A** -- Ask technical questions and get answers from the community
- **Ideas** -- Propose features and improvements
- **Show & Tell** -- Share your Dryade projects and integrations

**[Browse Discussions](https://github.com/DryadeAI/Dryade/discussions)**

## Contributing

Dryade is built by its community. There are many ways to contribute beyond code:

- Fix bugs and add features
- Improve documentation
- Create example projects
- Translate the interface
- Report issues and suggest improvements

**[Read the Contributing Guide](https://github.com/DryadeAI/Dryade/blob/main/CONTRIBUTING.md)**

## Examples

Get started quickly with our curated example projects:

- **Chat Quickstart** -- Connect an LLM and start chatting in 5 minutes
- **Agent Quickstart** -- Create an agent with tool calling
- **MCP Server Quickstart** -- Connect external tools via the Model Context Protocol

**[Browse Examples](https://github.com/DryadeAI/Dryade/tree/main/examples)**

## Code of Conduct

We are committed to providing a welcoming and respectful environment for everyone. Please be constructive in your interactions, respect different viewpoints, and help us maintain a positive community. Harassment, discrimination, and disruptive behavior are not tolerated.
